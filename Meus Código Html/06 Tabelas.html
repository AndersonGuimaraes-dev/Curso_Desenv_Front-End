Tags

<thead> é usada para agrupar conteúdo de um cabeçalho de tabela. Deve vir após as tags <caption> e <colgroup>, antes de elementos <tbody>, <tfoot> e <tr>.
<body> é usada para agrupar o conteúdo do corpo. Deve vir após tags <caption>, <colgroup> e <thead>.
<foot> é usada para agrupar conteúdo do rodapé. Deve vir após tags <caption>, <colgroup> e <thead>.
