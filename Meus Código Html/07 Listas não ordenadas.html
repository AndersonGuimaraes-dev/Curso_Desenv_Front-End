Tag de lista não ordenada: <ul>

Listas não ordenadas são lista que não precisam de uma numeração sequencial, seja por numeral arábico, romano ou por sequência de letras (a, b, c etc.).
"ul" significa unordered list.
"li"significa list item.

Atributos de <ul>

"type" - tipo de marcador de lista:
"circle" = círculo vazado;
"disc" = disco cheio;
"square" = quadrado;
"triangle" = triângulo (interface WebTV; nem todo navegador o suporta).

Esses marcadores estão deprecados. Devemos configurar o tpo de marcador usando a propriedade CSS "list-style-type".
